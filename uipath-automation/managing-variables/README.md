## Overview

This UiPath workflow demonstrates the use of variables, user input, data validation, exception handling, and basic business logic.

The workflow collects user information, validates a birth date, calculates the user's age, and displays formatted output.

---

## Workflow

User Input
        │
        ▼
Read Full Name
        │
        ▼
Read Birth Date
        │
        ▼
Validate Date Format
        │
        ▼
Calculate Age
        │
        ▼
Display Results
        │
        ▼
Log Information
