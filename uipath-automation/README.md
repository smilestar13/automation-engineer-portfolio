# UiPath Automation Portfolio

## Overview

This repository contains a collection of UiPath automation projects created while studying the UiPath Automation Developer learning path and solving practical automation scenarios.

Each project focuses on a specific automation concept and demonstrates how common business problems can be solved using UiPath workflows.

The portfolio covers everything from basic control flow and variables to Excel automation, DataTables, Forms, reusable workflows, and reporting.

> **Note**
>
> These projects were created for educational and portfolio purposes.
> All sample data, documents, forms, and business scenarios are fictional and intended solely to demonstrate automation techniques.
