# Interactive Web3 Node Installer

## Overview

This repository contains an interactive Bash installation wizard developed to simplify the deployment of a **Koii Network** node.

The primary goal of the project was to make Linux server deployment accessible to community members with little or no DevOps experience by automating the entire installation process through an interactive command-line interface.

Rather than following lengthy documentation or executing dozens of manual commands, users could complete the deployment by answering a few simple prompts.

---

## About the Project

**Koii Network** is a decentralized Web3 infrastructure platform focused on distributed computing and task execution.

When this installer was developed, the project had successfully raised **over $10 million** in funding from leading blockchain investors and maintained an active international community of node operators.

This installer was created as a community contribution to simplify onboarding and reduce common deployment errors.

---

## Purpose

The installer automates a complete Linux server setup required for running a network node.

It performs infrastructure preparation, dependency installation, environment configuration and service deployment while providing a guided multilingual user experience.

The script was designed to reduce manual work, improve reliability and provide a consistent deployment process across different Ubuntu servers.
